⚡ Solution Rapide!

Port 5506 - 3 minutes pour corriger

Port détecté

Vous accédez depuis http://127.0.0.1:5506

Ce port n'est pas encore configuré dans Google Cloud Console. Les étapes ci-dessous vont le corriger en 3 minutes!

1

Copier les deux origines

Vous devez ajouter ces deux origines à Google Cloud:

http://localhost:5506
http://127.0.0.1:5506
2

Ouvrir Google Cloud Console

Cliquez sur le bouton ci-dessous pour ouvrir Google Cloud:

3

Sélectionner le projet

En haut à gauche, assurez-vous que le projet sélectionné est:

Prestige Shop Express New
4

Cliquer sur l'ID client

Cherchez cette ligne dans la liste et cliquez dessus:

Prestige Shop Express New (type: Web application)
5

Scroll à la section "Origines JavaScript"

Scrollez vers le bas jusqu'à trouver la section:

Origines JavaScript autorisées (ou "Authorized JavaScript origins")
6

Ajouter les deux URIs

Cliquez sur "+ Ajouter URI" (ou "+ ADD URI")

Collez la première URL:

http://localhost:5506

Recliquez sur "+ Ajouter URI"

Collez la deuxième URL:

http://127.0.0.1:5506
7

Enregistrer les modifications

Cliquez sur le bouton "Enregistrer" (en bas) ou "SAVE"

✅ Vous devriez voir un message de confirmation
8

⏳ Attendez 3-5 minutes

Google met du temps à propager les changements. Patience!

Pendant ce temps, vous pouvez:

  • Prendre un ☕ café
  • Faire une pause
  • Vérifier que les URLs sont bien ajoutées (rechargez Google Cloud)
9

Vider le cache du navigateur

Appuyez sur: Ctrl + Shift + Delete

Puis:

  • Cochez "Cache"
  • Cochez "Cookies"
  • Cliquez "Effacer"
10

Rechargez la page

Appuyez sur: Ctrl + F5

Ou fermeture le navigateur et rouvrez-le.

✅ C'est terminé!
