# GUIDE D'UTILISATION DU CAROUSEL MÉDIA (IMAGES & VIDÉOS) POUR LES PRODUITS

## Introduction

Ce guide explique comment utiliser la nouvelle fonctionnalité de carrousel média pour les produits dans l'interface d'administration de Prestige Shop. Cette fonctionnalité permet d'associer plusieurs images et vidéos à un seul produit, offrant ainsi une expérience visuelle et interactive enrichie pour vos clients.

## Accès à la Gestion des Produits

1. Connectez-vous à l'interface d'administration
2. Cliquez sur "Gestion des Produits" dans le menu principal
3. Vous verrez la liste des produits existants avec leur carrousel média

## Ajouter un Nouveau Produit avec Images et Vidéos

### Étape 1 : Ouvrir le Formulaire
- Cliquez sur le bouton "Nouveau Produit"

### Étape 2 : Remplir les Informations de Base
- Remplissez les champs obligatoires : Nom, Prix, Catégorie
- Ajoutez une description détaillée du produit

### Étape 3 : Ajouter les Images
- Dans la section "Images du produit", entrez une URL d'image par ligne
- **Important** : La première image sera considérée comme l'image principale
- Exemple :
```
https://example.com/produit-image1.jpg
https://example.com/produit-image2.jpg
https://example.com/produit-image3.jpg
```

### Étape 4 : Ajouter les Vidéos
- Dans la section "Vidéos du produit", entrez une URL de vidéo par ligne
- Exemple :
```
https://example.com/produit-video1.mp4
https://example.com/produit-video2.mp4
```

### Étape 5 : Autres Informations
- Définissez le stock disponible
- Sélectionnez le statut du produit (actif, inactif, épuisé)
- Ajoutez les tailles et couleurs disponibles si nécessaire

### Étape 6 : Enregistrer
- Cliquez sur "Enregistrer" pour créer le produit

## Modifier un Produit Existant

1. Dans la liste des produits, cliquez sur le bouton "Modifier" pour le produit concerné
2. Mettez à jour les informations comme nécessaire
3. Modifiez les URLs d'images et de vidéos dans les champs dédiés
4. Cliquez sur "Enregistrer" pour sauvegarder les modifications

## Bonnes Pratiques

### Format des Fichiers
- **Images** : Utilisez des formats JPG ou PNG
- **Vidéos** : Utilisez des formats MP4, WebM ou OGG
- Préférez des images avec un rapport largeur/hauteur de 1:1 ou 4:3
- Optimisez la taille des fichiers pour un chargement rapide

### URLs des Médias
- Assurez-vous que les URLs sont accessibles publiquement
- Utilisez des services d'hébergement de médias fiables
- Vérifiez régulièrement que les liens sont toujours valides

### Nombre de Médias
- Il est recommandé d'utiliser entre 2 et 8 médias (images + vidéos) par produit
- Trop de médias peuvent ralentir le chargement de la page

## Gestion des Erreurs

### Média Manquant
- Si une image ne se charge pas, une image de remplacement sera automatiquement affichée
- L'image de remplacement par défaut est `/images/prestige shop.png`
- Si une vidéo ne se charge pas, elle sera ignorée dans le carrousel

### Problèmes de Chargement
- Vérifiez que les URLs sont corrects
- Assurez-vous que les médias sont accessibles sans authentification
- Contactez le support technique si le problème persiste

## Aperçu du Carrousel

Dans la liste des produits :
- Chaque produit affiche un carrousel combinant images et vidéos dans la colonne "Image"
- Pour les produits avec plusieurs médias, des flèches de navigation et des points indicateurs sont visibles
- Un badge indique le nombre de vidéos présentes dans le carrousel
- Cliquez sur les flèches ou les points pour naviguer entre les médias
