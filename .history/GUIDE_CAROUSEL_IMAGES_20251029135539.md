# GUIDE D'UTILISATION DU CAROUSEL D'IMAGES POUR LES PRODUITS

## Introduction

Ce guide explique comment utiliser la nouvelle fonctionnalité de carrousel d'images pour les produits dans l'interface d'administration de Prestige Shop. Cette fonctionnalité permet d'associer plusieurs images à un seul produit, offrant ainsi une meilleure expérience visuelle pour vos clients.

## Accès à la Gestion des Produits

1. Connectez-vous à l'interface d'administration
2. Cliquez sur "Gestion des Produits" dans le menu principal
3. Vous verrez la liste des produits existants avec leur carrousel d'images

## Ajouter un Nouveau Produit avec Plusieurs Images

### Étape 1 : Ouvrir le Formulaire
- Cliquez sur le bouton "Nouveau Produit"

### Étape 2 : Remplir les Informations de Base
- Remplissez les champs obligatoires : Nom, Prix, Catégorie
- Ajoutez une description détaillée du produit

### Étape 3 : Ajouter les Images
- Dans la section "Images du produit", entrez une URL d'image par ligne
- **Important** : La première image sera considérée comme l'image principale
- Exemple :
```
https://example.com/produit-image1.jpg
https://example.com/produit-image2.jpg
https://example.com/produit-image3.jpg
```

### Étape 4 : Autres Informations
- Définissez le stock disponible
- Sélectionnez le statut du produit (actif, inactif, épuisé)
- Ajoutez les tailles et couleurs disponibles si nécessaire

### Étape 5 : Enregistrer
- Cliquez sur "Enregistrer" pour créer le produit

## Modifier un Produit Existant

1. Dans la liste des produits, cliquez sur le bouton "Modifier" pour le produit concerné
2. Mettez à jour les informations comme nécessaire
3. Modifiez les URLs d'images dans le champ dédié
4. Cliquez sur "Enregistrer" pour sauvegarder les modifications

## Bonnes Pratiques

### Format des Images
- Utilisez des images au format JPG ou PNG
- Préférez des images avec un rapport largeur/hauteur de 1:1 ou 4:3
- Optimisez la taille des fichiers pour un chargement rapide

### URLs des Images
- Assurez-vous que les URLs sont accessibles publiquement
- Utilisez des services d'hébergement d'images fiables
- Vérifiez régulièrement que les liens sont toujours valides

### Nombre d'Images
- Il est recommandé d'utiliser entre 2 et 5 images par produit
- Trop d'images peuvent ralentir le chargement de la page

## Gestion des Erreurs

### Image Manquante
- Si une image ne se charge pas, une image de remplacement sera automatiquement affichée
- L'image de remplacement par défaut est `/images/prestige shop.png`

### Problèmes de Chargement
- Vérifiez que les URLs sont corrects
- Assurez-vous que les images sont accessibles sans authentification
- Contactez le support technique si le problème persiste

## Aperçu du Carrousel

Dans la liste des produits :
- Chaque produit affiche un carrousel d'images dans la colonne "Image"
- Pour les produits avec plusieurs images, des flèches de navigation et des points indicateurs sont visibles
- Cliquez sur les flèches ou les points pour naviguer entre les images
