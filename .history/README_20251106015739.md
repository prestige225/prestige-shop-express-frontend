# Prestige Shop - Backend

## Déploiement sur Render

Pour déployer ce backend sur Render, suivez ces étapes :

1. Créez un compte sur [Render](https://render.com/) si vous n'en avez pas déjà un.

2. Créez un nouveau service Web :
   - Cliquez sur "New" puis "Web Service"
   - Connectez votre dépôt GitHub contenant ce code
   - Render détectera automatiquement qu'il s'agit d'une application Python grâce au fichier `requirements.txt`

3. Configuration du service :
   - **Name**: Donnez un nom à votre service (ex: prestige-shop-backend)
   - **Runtime**: Python 3
   - **Build Command**: `pip install -r requirements.txt`
   - **Start Command**: `python server_fixed.py`
   - **Plan**: Choisissez le plan gratuit ou payant selon vos besoins

4. Variables d'environnement :
   - Assurez-vous que les variables d'environnement pour la base de données sont correctement configurées dans le tableau de bord Render

5. Déploiement :
   - Cliquez sur "Create Web Service"
   - Render construira et déploiera automatiquement votre application

## Configuration de la base de données

Le backend utilise une base de données MySQL. Vous devez configurer les variables d'environnement suivantes dans Render :

- `DB_HOST`: L'hôte de votre base de données
- `DB_USER`: Le nom d'utilisateur de votre base de données
- `DB_PASSWORD`: Le mot de passe de votre base de données
- `DB_NAME`: Le nom de votre base de données
- `DB_PORT`: Le port de votre base de données (généralement 3306)

## Fichiers de configuration

- `render.yaml` : Fichier de configuration pour le déploiement sur Render
- `requirements.txt` : Liste des dépendances Python
- `runtime.txt` : Version de Python à utiliser
