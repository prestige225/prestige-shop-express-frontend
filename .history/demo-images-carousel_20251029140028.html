Instructions d'utilisation

  1. Pour ajouter plusieurs images à un produit :
    • Allez dans la section "Gestion des Produits" de l'admin
    • Cliquez sur "Nouveau Produit" ou "Modifier" un produit existant
    • Dans le champ "Images du produit", entrez une URL d'image par ligne
    • La première image sera considérée comme l'image principale
  2. Format des URLs :
    • Assurez-vous que les URLs pointent vers des images valides
    • Les formats supportés sont : JPG, PNG, GIF
    • Exemple : 
      https://example.com/image1.jpg
https://example.com/image2.jpg
https://example.com/image3.jpg
  3. Gestion des images manquantes :
    • Si une image ne se charge pas, une image de remplacement sera automatiquement utilisée
    • Vous pouvez modifier l'image de remplacement dans les paramètres du site
